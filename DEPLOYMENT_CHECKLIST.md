## 배포 후 테스트 플로우

### 사용자 회원가입 → 로그인 → 의견 작성 플로우
```
1. 회원가입 페이지 (/signup)
   - 이메일 입력
   - 비밀번호 입력
   - 회원가입 버튼 클릭
   - ✅ 회원가입 성공 확인

2. 로그인 페이지 (/login)
   - 이메일 입력
   - 비밀번호 입력
   - 로그인 버튼 클릭
   - ✅ 로그인 성공 확인

3. 팀 선택 페이지 (/team-selection)
   - 응원팀 선택
   - 계속하기 버튼 클릭
   - ✅ 팀 선택 저장 확인

4. 구단 페이지 (/app)
   - 의견 목록 로드 확인
   - 의견 작성 버튼 클릭

5. 의견 작성 페이지 (/app/create)
   - 카테고리 선택
   - 제목 입력
   - 내용 입력
   - 작성 완료 버튼 클릭
   - ✅ 의견 저장 확인

6. 구단 페이지 (/app)
   - ✅ 새로운 의견이 피드에 표시되는지 확인

7. 의견 상세 페이지 (/app/opinion/[id])
   - 의견 내용 확인
   - 좋아요 버튼 클릭
   - ✅ 좋아요 수 증가 확인
   - 댓글 입력
   - 댓글 등록 버튼 클릭
   - ✅ 댓글 저장 확인

8. 설문 페이지 (/app/surveys)
   - 설문 목록 로드 확인
   - 설문 선택
   - 옵션 선택
   - 제출 버튼 클릭
   - ✅ 설문 응답 저장 확인

9. 프로필 페이지 (/app/profile)
   - 사용자 정보 확인
   - 활동 통계 확인
   - ✅ 작성한 의견 수, 참여한 설문 수 업데이트 확인
```
