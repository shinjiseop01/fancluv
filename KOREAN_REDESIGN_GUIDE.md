# FANCLUV 한국어 중심 서비스 재설계 가이드

## 📋 개요

FANCLUV가 **한국 팬 대상 서비스**로 완전히 재설계되었습니다. 모든 UI가 한국어 중심으로 구성되었으며, 실제 서비스 수준의 대시보드와 온보딩 흐름을 제공합니다.

---

## 🎯 새로운 온보딩 흐름

```
1️⃣ 초기 접속
   ↓
   로그인 페이지
   - Split Screen 레이아웃
   - K리그 경기장 이미지 (좌측)
   - FANCLUV 로고 + 로그인 폼 (우측)
   - 비밀번호 찾기 기능
   - 회원가입 링크
   ↓
2️⃣ 로그인/회원가입 완료
   ↓
   localStorage에 사용자 정보 저장
   ↓
3️⃣ 응원 구단 선택
   ↓
   K리그 12개 구단 카드
   - FC서울, 울산HD, 전북현대, 강원FC
   - 포항스틸러스, 인천유나이티드, FC안양
   - 제주유나이티드, 부천FC1995
   - 대전하나시티즌, 김천상무, 광주FC
   ↓
4️⃣ 메인 대시보드
   ↓
   프리미엄 한국 스포츠 플랫폼 대시보드
```

## 📞 지원

### 문제 해결

**Q: 로그인 페이지가 영어로 표시됩니다**
A: localStorage에서 언어 설정을 확인하세요. 개발자 도구 → Application → localStorage에서 `language` 값을 확인하세요.

**Q: 팀 선택 후 대시보드로 이동하지 않습니다**
A: 브라우저 콘솔에서 오류를 확인하세요. localStorage에 `selectedTeam` 값이 저장되었는지 확인하세요.

**Q: 모바일에서 레이아웃이 깨집니다**
A: 브라우저 캐시를 삭제하고 새로고침하세요. (Cmd+Shift+Delete)
