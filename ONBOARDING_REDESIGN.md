# FANCLUV 온보딩 흐름 전면 개편 가이드

## 📋 개요

FANCLUV의 온보딩 흐름이 F1 Fan Voice 수준의 프리미엄 스포츠 플랫폼으로 전면 개편되었습니다.

---

## 🎯 새로운 온보딩 흐름

### 1단계: 초기 접속
```
https://fancluv.vercel.app/
↓
로그인 페이지 (index.tsx)
```

**특징:**
- 좌측: K리그 경기장 대형 이미지
- 우측: FANCLUV 로고, 슬로건, 로그인 폼
- 회원가입 링크 제공
- 우측 상단: 언어 선택 (한국어/영어)

### 2단계: 로그인 또는 회원가입
```
로그인 (/index.tsx) 또는 회원가입 (/signup.tsx)
↓
계정 생성/로그인 완료
↓
localStorage에 사용자 정보 저장
```

**특징:**
- 동일한 Split Screen 레이아웃
- 다국어 지원 (한국어/영어)
- 반응형 디자인 (모바일/태블릿/데스크톱)

### 3단계: 응원 구단 선택
```
로그인/회원가입 완료
↓
팀 선택 페이지 (/team-selection.tsx)
↓
K리그 12개 구단 중 선택
↓
localStorage에 선택 구단 저장
```

**특징:**
- K리그 12개 구단 표시
- 카드 전체 클릭 가능
- Hover 애니메이션 (위로 이동, 그림자 강화)
- 다국어 지원

### 4단계: 메인 대시보드
```
구단 선택 완료
↓
메인 대시보드 (/app/dashboard.tsx)
↓
설문, 의견, 통계, 활동 확인
```

**특징:**
- 프리미엄 대시보드 UI
- 최신 설문 표시
- 인기 의견 게시판
- 참여 통계
- 구단 뉴스
- 내 활동 현황

## 📞 지원

### 문제 해결

**Q: 로그인 후 팀 선택 페이지가 표시되지 않음**
```
A: localStorage에 user 정보가 저장되었는지 확인
   개발자 도구 → Application → Local Storage 확인
```

**Q: 언어 변경이 작동하지 않음**
```
A: 브라우저 캐시 삭제 후 다시 시도
   또는 개발자 도구에서 localStorage 확인
```

**Q: 모바일에서 레이아웃이 깨짐**
```
A: 뷰포트 설정 확인
   <meta name="viewport" content="width=device-width, initial-scale=1">
```
